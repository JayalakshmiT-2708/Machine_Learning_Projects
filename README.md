# Machine Learning Projects

## Overview

This repository contains Machine Learning projects developed to apply data analysis, feature engineering, model training, and predictive modeling techniques on real-world datasets.

The projects demonstrate the complete machine learning workflow, including data preprocessing, exploratory data analysis, model building, evaluation, and prediction.

---

## Projects Included

### 1. Insurance Charge Prediction

#### Description
This project predicts medical insurance charges based on various factors such as age, gender, BMI, number of children, smoking habits, and region.

#### Objectives
- Analyze factors affecting insurance costs
- Perform data preprocessing and feature encoding
- Train machine learning models for cost prediction
- Evaluate model performance using regression metrics

#### Concepts Used
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Algorithms
- Model Evaluation

#### Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
